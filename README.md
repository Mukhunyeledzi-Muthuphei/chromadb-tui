# Chroma GUI (TUI)

A Textual TUI for browsing a ChromaDB instance.

## Installation

To install the Chroma GUI, you need Python 3.8+ and `pip`.

1.  Clone this repository (or download the source code).
2.  Navigate to the project directory.
3.  Install the project in editable mode:

    ```bash
    pip install -e .
    ```

    This will install the necessary dependencies and create the `chroma-gui` command-line executable.

## Usage

Once installed, you can run the TUI from your terminal:

```bash
chroma-gui
```
